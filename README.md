# לוח תשואות אג"ח ממשלתיות

אתר סטטי קטן המציג טבלה עם תשואות (ריבית) אג"ח ממשלתיות של מדינות מרכזיות וישראל, לכל תקופה (מח"מ) - 3 חודשים, 1, 2, 5, 10 ו-30 שנה.

## הרצה

לא נדרש שום build או התקנה - פשוט פתחו את `index.html` בדפדפן.

לחלופין, להרצה עם שרת מקומי (מומלץ אם יש בעיות CORS עתידיות):

```
npx serve .
```

## מבנה הפרויקט

```
index.html      - מבנה העמוד
css/style.css   - עיצוב
js/data.js      - נתוני התשואות (כאן מעדכנים!)
js/app.js       - בניית הטבלה, מיון וצביעת מדרגי חום
```

## עדכון הנתונים

כל הנתונים נמצאים בקובץ [`js/data.js`](js/data.js) ומוגדרים כ-**נתוני דמו להמחשה**.
לעדכון:

1. עדכנו את `asOf` (התאריך שמוצג בתחתית העמוד).
2. עדכנו את מערך `yields` של כל מדינה - אחוזי **תשואה שנתית (אנצואלית)**, גם
   עבור תקופות קצרות משנה, לפי הסדר שמוגדר ב-`periods`.
3. אפשר להוסיף/להוריד מדינות על ידי הוספה/הסרה של אובייקט במערך `countries`
   (עם `name`, `code` דו-אותיות, `source`/`sourceName` ו-`yields` באותו אורך
   כמו `periods`).

### מקורות נתונים

כל ערך תשואה בטבלה הוא קישור (`source`) למקור עדכני:

- **7 המדינות האחרות** - העמוד הרלוונטי ב-[World Government Bonds](https://www.worldgovernmentbonds.com/).
- **ישראל** - עמוד ["אג"ח ומק"מ - תשואות לפדיון" של בנק ישראל](https://www.boi.org.il/roles/statistics/makamandbonds/yield/),
  שמכיל גם את תשואות המק"מ (לתקופות של עד שנה) וגם את עקום התשואות של אג"ח ממשלתי
  (לתקופות הארוכות יותר).

שימו לב: הערכים בטבלה הם נתוני דמו ולכן לא יתאימו בהכרח לערכים העדכניים במקורות.

## פיצ'רים

- מיון לפי כל עמודה (לחיצה על כותרת העמודה).
- צביעת מדרגי חום (ירוק = תשואה נמוכה יותר, אדום = תשואה גבוהה יותר) בתוך כל עמודה.
- כל ערך תשואה הוא קישור למקור הנתונים העדכני (נפתח בכרטיסייה חדשה).
- עיצוב רספונסיבי עם גלילה אופקית במובייל ועמודת מדינה "דביקה".
