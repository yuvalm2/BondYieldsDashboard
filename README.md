# לוח תשואות אג"ח ממשלתיות

אתר המציג טבלה עם תשואות (ריבית שנתית) אג"ח ממשלתיות של מדינות מרכזיות וישראל, לכל
תקופה (מח"מ) - 3 חודשים, 1, 2, 5, 10 ו-30 שנה. חלק מהתאים **חיים** (נשלפים אוטומטית
ממקור חינמי בכל טעינה), וחלק **דמו** (ערכים קבועים) - כל תא מתויג בבירור בטבלה.

## הרצה

יש שרת Node קטן (בלי תלויות חיצוניות) שמגיש את האתר וגם מביא את הנתונים החיים:

```
node server.js
```

ואז לפתוח את http://localhost:8080 (אפשר לשנות פורט עם משתנה הסביבה `PORT`).

## מבנה הפרויקט

```
server.js            - שרת Node: מגיש את האתר + /api/yields (מיזוג חי+דמו, עם מטמון לשעה)
data/demo-data.js     - נתוני בסיס/פולבק (כאן מעדכנים ערכי דמו ומוסיפים/מסירים מדינות)
data/live-sources.js  - פונקציות השליפה החיה (ארה"ב, קנדה, גרמניה, בריטניה, יפן, צרפת, איטליה)
index.html            - מבנה העמוד
css/style.css         - עיצוב
js/app.js             - fetch מה-API, בניית הטבלה, מיון וצביעת מדרגי חום
```

## מקורות נתונים

| מדינה | חי/דמו | מקור | תדירות |
|---|---|---|---|
| ארה"ב | חי (כל התקופות) | [Treasury.gov](https://home.treasury.gov/resource-center/data-chart-center/interest-rates) - עקום תשואות יומי | יומי |
| גרמניה | חי (כל התקופות, עקום אזור האירו AAA כקירוב) | [ECB Data Portal](https://data-api.ecb.europa.eu/) | יומי |
| בריטניה, יפן, צרפת, איטליה | חי (כל התקופות) | [World Government Bonds](https://www.worldgovernmentbonds.com/) - API פנימי (wp-json), ראו הערה למטה | יומי |
| קנדה | חי (כל התקופות: 2/5/10/30 שנה מבנק קנדה, 3 חודשים/שנה מ-World Government Bonds) | [Bank of Canada Valet API](https://www.bankofcanada.ca/valet/) + World Government Bonds | יומי |
| ישראל | דמו (לא נמצא API פומבי) | [בנק ישראל - תשואות לפדיון](https://www.boi.org.il/roles/statistics/makamandbonds/yield/) (מק"מ + אג"ח) | - |

תאי "דמו" אינם חסרי מקור - הם מקושרים לאותו מקור, רק שהערך עצמו מוזן ידנית
בקובץ `data/demo-data.js` ולא נשלף אוטומטית (כי לא נמצא API חינמי וללא מפתח לאותו צירוף
מדינה/תקופה). כיום ישראל היא היחידה שמוצגת כ"דמו" במלואה.

**על ה-API של World Government Bonds:** האתר הוא Single Page App שמרנדר את הטבלאות
ב-JS, אבל מצאנו שהוא שולף את הנתונים מ-endpoint פנימי נקי -
`POST https://www.worldgovernmentbonds.com/wp-json/country/v1/main` - שמחזיר JSON
(כולל טבלת העקום כ-HTML מובנה בתוכו) בתשובה לכותרות `Referer`/`Origin` תקינות, בלי
צורך בדפדפן headless. ראו `fetchWGB()` ב-`data/live-sources.js`.

### עדכון נתוני הדמו

ב-[`data/demo-data.js`](data/demo-data.js):

1. עדכנו את `asOf` (תאריך ערכי הדמו, מוצג בתחתית העמוד).
2. עדכנו את מערך `yields` של כל מדינה - אחוזי **תשואה שנתית (אנצואלית)**, גם
   עבור תקופות קצרות משנה, לפי הסדר שמוגדר ב-`periods`.
3. אפשר להוסיף/להוריד מדינות על ידי הוספה/הסרה של אובייקט במערך `countries`
   (עם `name`, `code` דו-אותיות, `source`/`sourceName` ו-`yields` באותו אורך
   כמו `periods`).

### הוספת מקור חי למדינה נוספת

ב-[`data/live-sources.js`](data/live-sources.js) מוסיפים פונקציית `fetchXxx`
שמחזירה `{ asOf: "YYYY-MM-DD", values: { [periodIndex]: number } }` (אינדקס
0-5 לפי `periods`), ורושמים אותה ב-`module.exports` עם קוד המדינה (כמו `US`,
`CA`, `DE`, `GB`, `JP`, `FR`, `IT`). אפשר להחזיר רק חלק מהאינדקסים אם אין מקור
לכל התקופות - השאר יישארו "דמו" אוטומטית.

אם מדינה חדשה קיימת ב-[World Government Bonds](https://www.worldgovernmentbonds.com/),
הדרך הקלה ביותר היא `fetchWGB(symbol, urlPage)` הקיימת - רק צריך למצוא את ה-`SYMBOL`
המספרי של המדינה (גלוי במשתנה `jsGlobalVars` במקור הדף) ואת `urlPage` (slug ב-URL).
לישראל לא מצאנו API פומבי (לבנק ישראל יש "Fusion Data Browser" מבוסס JS בלי API
שזיהינו), ובכל מקרה לא משתמשים ב-World Government Bonds בשבילה כי המקור המוצג
לישראל הוא בנק ישראל (מק"מ+אג"ח) ולא World Government Bonds.

## פיצ'רים

- כל תא מתויג **חי** (נשלף בזמן אמת, עם תאריך עדכון ב-tooltip) או **דמו** (ערך קבוע).
- מיון לפי כל עמודה (לחיצה על כותרת העמודה).
- צביעת מדרגי חום (אדום = תשואה נמוכה יותר, ירוק = תשואה גבוהה יותר) בתוך כל עמודה.
- כל ערך תשואה הוא קישור למקור הנתונים (נפתח בכרטיסייה חדשה).
- מטמון בזיכרון של שעה לנתונים החיים, עם נפילה חזרה לערך האחרון שהצליח (או לדמו) אם מקור חי נכשל.
- בקשות שמגיעות בזמן שהמטמון פג ורענון כבר מתבצע "מתחברות" לאותו רענון במקום לשלוח בקשה כפולה למקור החיצוני.
- עיצוב רספונסיבי עם גלילה אופקית במובייל ועמודת מדינה "דביקה".
